# Microarray Expression Data Analysis Practice

This repository contains practice exercises for analyzing microarray expression data. The goal is to explore and understand techniques used in bioinformatics for microarray data analysis.

## Data Reference and Learning sources

The dataset used in this practice is from:

- **Zia Khan, Urooj Syed, Mackenzie Hsu & Christopher Howlett (2025).**  
*Age-related vascular remodeling in bone marrow.* BioStudies, E-MTAB-15951.  
Retrieved from: [https://www.ebi.ac.uk/biostudies/ArrayExpress/studies/E-MTAB-15951](https://www.ebi.ac.uk/biostudies/ArrayExpress/studies/E-MTAB-15951)

## Overview

The exercises include:

- Data preprocessing and normalization
- Differential expression analysis
- Visualization of microarray data
- Interpretation of biological results

This repository is intended for learning and practice purposes only.
